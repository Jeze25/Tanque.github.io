Tanque

Proyecto desplegado para GitHub Pages. Esta página actúa como la entrada principal del portafolio y el mini juego de combate tanque, con un diseño claro y útil para visitantes.

¿Qué es este proyecto?

Tanque es un proyecto de demostración para una experiencia interactiva de combate en navegador. Aquí puedes presentar el juego y explicar cómo se probará en GitHub Pages.

Usar GitHub Pages

El sitio está listo para publicarse en GitHub Pages usando el archivo index.html en la raíz del repositorio. Si el repositorio se llama Tanque.github.io, la publicación queda disponible automáticamente en https://Jeze25.github.io/.
