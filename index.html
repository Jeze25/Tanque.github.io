Tanquera

Juego interactivo de combate tanque para GitHub Pages. Muévete con WASD, dispara con Espacio y destruye los drones enemigos.

Bienvenido

Este proyecto ahora es una página interactiva lista para GitHub Pages. Usa los controles del panel de juego para probar una demo en el navegador sin necesidad de instalación.

Haz clic en Jugar para entrar al campo de batalla y ver cómo el tanque se mueve, dispara y esquiva los drones enemigos.

HTML + JS Proyecto 100% estático
GitHub Pages Despliegue instantáneo
Controles WASD + Espacio
Interactividad Canvas en vivo

Zona de Juego

  • W Avanzar
  • A Izquierda
  • S Retroceder
  • D Derecha
  • ␣ Disparar
